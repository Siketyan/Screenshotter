# Screenshotter
Windows標準のSnipping Tool風にスクリーンショットを取得し、メッセージとともにツイートします。  
このソフトウェアは、[come25136/MyScreenshotAssistant2](https://github.com/come25136/MyScreenshotAssistant2)をインスパイアしています。
